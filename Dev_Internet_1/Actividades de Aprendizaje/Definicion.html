Definicion de HTML

HTML (Lenguaje de Marcado de Hipertexto) es un código que se utiliza para estructurar y mostrar una página web y su contenido. Es un lenguaje descriptivo que define la estructura básica de una página web y el código para plasmar el contenido.

El HTML es el componente más básico de la web y se compone de un conjunto de etiquetas que el navegador interpreta. Estas etiquetas permiten definir el texto, imágenes, listas, tablas, videos, entre otros elementos que componen una página web.
