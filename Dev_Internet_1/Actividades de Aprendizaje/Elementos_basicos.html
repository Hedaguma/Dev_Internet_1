Elementos basicos de HTML

<html>: Define el principio y el final de un documento HTML.

<head>: Se utiliza para contener información sobre el documento, como el título de la página o el enlace a una hoja de estilo.

<body>: Contiene el texto y contenido visible de una página web, como imágenes, enlaces, etc.

<p>: Crea un párrafo de texto.

<img>: Se utiliza para insertar imágenes en la página.

<a>: Define un enlace que puede conducir a otra página o recurso.

<div>: Se usa para agrupar otros elementos HTML y organizar la página.

<span>: Se usa para aplicar estilos o realizar operaciones específicas. Se aplica a un fragmento de texto dentro de un párrafo o sección.

<title>: Define el título del documento, tal como se muestra en la barra de título del navegador.

<h1> - <h6>: Se utilizan para crear un título. <h1> es el más importante y <h6> es el menos importante.

<ul> <ol> <li>: Se utiliza para crear una lista desordenada (<ul>) o una lista ordenada (<ol>). Los elementos de la lista se colocan dentro de <li>.

<form>: Define un formulario en HTML en el que los usuarios pueden enviar información.

<input>: Permite ingresar datos del usuario en el formulario, como texto, contraseñas, botones, etc.
